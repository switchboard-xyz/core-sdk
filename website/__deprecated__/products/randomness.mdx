Offers on-chain publication of pseudo-random values, which can be used to
generate unpredictable outcomes in applications. An on-chain lottery system
integrated into a DeFi platform can use Switchboard pSeudorandomness to draw
winning numbers fairly, ensuring a transparent and tamper-proof lottery process
for users.

## Use Cases

- On-chain lotteries and raffles
- Generating in game assets in a verifiable manner
- Generating NFT traits
