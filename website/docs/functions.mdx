# Functions

:::info

Switchboard Functions are currently in **BETA** on Solana devnet. Find us on
Discord to start integrating.

:::

## Overview

Switchboard Functions use Trusted Execution Environments (TEEs) to provide
verifiable off-chain code execution. This allows a smart contract developer to
automate tasks or provide async operations in their smart contracts.
