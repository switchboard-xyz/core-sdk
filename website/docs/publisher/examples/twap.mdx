# TWAP

Aggregators can be initialized with a history buffer to store a buffer of
accepted results. The TWAP task will parse the history buffer, filter the
results within the specified period, and return the average.

<details>
<summary>

### BTC/USD 15min TWAP

</summary>

```json title="BTC_USD_15m_TWAP.json"
{
  "name": "BTC/USD 15min TWAP",
  "tasks": [
    {
      "twapTask": {
        /* Mainnet BTC/USD Feed */
        "aggregatorPubkey": "8SXvChNYFhRq4EZuZvnhjrB3jJRQCv4k3P4W6hesH3Ee",
        "period": 900
      }
    }
  ]
}
```

</details>
