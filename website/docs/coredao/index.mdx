# Switchboard x EVM

> A collection of libraries and examples for interacting with Switchboard on EVM
> chains.

[![NPM Badge](https://img.shields.io/github/package-json/v/switchboard-xyz/sbv2-evm?color=red&filename=javascript%2Fevm.js%2Fpackage.json&label=%40switchboard-xyz%2Fevm.js&logo=npm)](https://www.npmjs.com/package/@switchboard-xyz/evm.js)

</div>

## Getting Started

To get started, clone the
[sbv2-evm](https://github.com/switchboard-xyz/sbv2-evm) repository.

```bash
git clone https://github.com/switchboard-xyz/sbv2-evm
```

Then install the dependencies

```bash
cd sbv2-evm
pnpm install
pnpm build
```

## Clients

| **Lang**   | **Name**                                            | **Description**                                                                   |
| ---------- | --------------------------------------------------- | --------------------------------------------------------------------------------- |
| Solidity   | [ISwitchboard](coredao/sdk/solidity/ISwitchboard)   | A Solidity contract to read a Switchboard data feed.                              |
| Solidity   | [ILegacyOracle](coredao/sdk/solidity/ILegacyOracle) | A Solidity contract to read a Switchboard data feed with a legacy oracle adapter. |
| Javascript | [@switchboard-xyz/evm.js](coredao/sdk/javascript/)  | A Typescript client to interact with Switchboard on EVM chains.                   |
