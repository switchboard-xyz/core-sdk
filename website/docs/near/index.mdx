## Getting Started

To get started, clone the
[sbv2-near](https://github.com/switchboard-xyz/sbv2-near) repository.

```bash
git clone https://github.com/switchboard-xyz/sbv2-near
```

Then install the dependencies

```bash
cd sbv2-near
pnpm install
pnpm build
```

## Clients

| **Lang**   | **Name**                                         | **Description**                                                                      |
| ---------- | ------------------------------------------------ | ------------------------------------------------------------------------------------ |
| Rust       | [sbv2-near](near/sdk/rust/)                      | A Rust crate to deserialize and read a Switchboard data feed within a NEAR contract. |
| Javascript | [@switchboard-xyz/near.js](near/sdk/javascript/) | A Typescript client to interact with Switchboard on NEAR.                            |
