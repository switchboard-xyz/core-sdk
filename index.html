Switchboard

A multi-chain, permissionless oracle protocol delivering secure, reliable, and trustless off-chain data to on-chain smart contracts across Aptos, CoreDAO, NEAR, Solana, Starknet, and Sui. Switchboard empowers developers to build robust and feature-rich dApps with confidence and flexibility.

Switchboard V3 brings secure, off-chain computation, enabling developers to harness the power of trusted execution environments with verifiable, tamper-proof code execution, fostering trust, security, and endless possibilities for summoning data on-chain.
