Developer Resources

The switchboard-xyz/sbv2-core repo contains the chain agnostic code such as the CLI and OracleJob protobufs along with a git submodule for each chain specific SDK.

To get started, clone the sbv2-core repo along with the submodules.

Common​

git clone --recurse-submodules https://github.com/switchboard-xyz/sbv2-core.git
cd sbv2-core
common

Common type definitions used across chains such as the OracleJob protobuf

cli

Switchboard command line interface

Solana​

git clone https://github.com/switchboard-xyz/sbv2-solana.git
cd sbv2-solana
switchboard-v2

View the Solana rust client

solana.js

View the Solana typescript client

switchboardpy

View Solana python client

Aptos​

git clone https://github.com/switchboard-xyz/sbv2-aptos.git
cd sbv2-aptos
switchboard

View the Aptos move client

aptos.js

View the Aptos typescript client

Near​

git clone https://github.com/switchboard-xyz/sbv2-near.git
cd sbv2-near
sbv2-near

View the Near rust client

near.js

View the Near typescript client
