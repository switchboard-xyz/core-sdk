The Network

The Switchboard program is deployed on multiple chains, each with a native deployment with isolated oracle queues where the queue authority can configure the settings to influence its security.

Switchboard currently supports a native deployment on the following chains and networks:

  •  Solana
  •  Ethereum
  •  Core
  •  Arbitrum
  •  Optimism
  •  Base
  •  Aurora
  •  Aptos
  •  Sui
