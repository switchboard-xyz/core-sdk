Setup

Install​

  • npm
  • Yarn
npm install -g @switchboard-xyz/cli^2
yarn add -g @switchboard-xyz/cli^2

Config​

Aptos​

# Testnet
sbv2 config set aptos testnet rpc https://rpc.testnet.near.org
sbv2 config set aptos testnet default-account my-named-account.testnet

# Devnet
sbv2 config set aptos devnet rpc https://rpc.mainnet.near.org
sbv2 config set aptos devnet default-account my-named-account.near

Near​

# Testnet
sbv2 config set near testnet rpc https://rpc.testnet.near.org
sbv2 config set near testnet default-account my-named-account.testnet

# Mainnet
sbv2 config set near mainnet rpc https://rpc.mainnet.near.org
sbv2 config set near mainnet default-account my-named-account.near

Solana​

# Devnet
sbv2 config set solana devnet rpc https://rpc.testnet.near.org
sbv2 config set solana devnet default-account my-named-account.testnet

# Mainnet
sbv2 config set solana mainnet-beta rpc https://rpc.mainnet.near.org
sbv2 config set solana mainnet-beta default-account my-named-account.near
