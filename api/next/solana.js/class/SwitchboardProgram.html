  • SwitchboardProgram

Wrapper class for the Switchboard anchor Program.

This class provides an interface to interact with the Switchboard program on the Solana network. It allows you to load the program, create and initialize connection objects, and interact with Switchboard accounts.

Basic usage example:

import { Connection } from "@solana/web3.js";
import { SwitchboardProgram, TransactionObject } from '@switchboard-xyz/solana.js';

const program = await SwitchboardProgram.load(
   "mainnet-beta",
   new Connection("https://api.mainnet-beta.solana.com"),
   payerKeypair
);

const txn = new TransactionObject(program.walletPubkey, [], []);
const txnSignature = await program.signAndSend(txn);

loadPermissioned

  • loadPermissioned(): Promise<{ crank: CrankAccountData; crankAccount: CrankAccount; queue: OracleQueueAccountData; queueAccount: QueueAccount }>

  • Load the Switchboard Labs permissionled Queue for either devnet or mainnet. The permissioned queue has the following permissions:

    • unpermissionedFeedsEnabled: False
    • unpermissionedVrfEnabled: False
    • enableBufferRelayers: False

    Note: The queue authority must grant AggregatorAccounts PERMIT_ORACLE_QUEUE_USAGE and VrfAccounts PERMIT_VRF_REQUESTS permissions before joining the queue and requesting oracle updates. BufferRelayerAccounts are disabled.

    Returns Promise<{ crank: CrankAccountData; crankAccount: CrankAccount; queue: OracleQueueAccountData; queueAccount: QueueAccount }>

loadPermissionless

  • loadPermissionless(): Promise<{ crank: CrankAccountData; crankAccount: CrankAccount; queue: OracleQueueAccountData; queueAccount: QueueAccount }>

  • Load the Switchboard Labs permissionless Queue for either devnet or mainnet. The permissionless queue has the following permissions:

    • unpermissionedFeedsEnabled: True
    • unpermissionedVrfEnabled: True
    • enableBufferRelayers: False

    Note: AggregatorAccounts and VrfAccounts do not require permissions to join this queue. BufferRelayerAccounts are disabled.

    Returns Promise<{ crank: CrankAccountData; crankAccount: CrankAccount; queue: OracleQueueAccountData; queueAccount: QueueAccount }>

      The Solana connection object used to connect to an RPC node.

    • payer: any = READ_ONLY_KEYPAIR

      Optional, payer keypair used to pay for on-chain transactions (defaults to READ_ONLY_KEYPAIR).

    • optionalprogramId: any

    Returns Promise<SwitchboardProgram>

    The SwitchboardProgram instance used to create and interact with Switchboard accounts.

    Basic usage example:

    import * as anchor from "@coral-xyz/anchor";
    import { Connection } from "@solana/web3.js";
    import { AnchorWallet, SwitchboardProgram, TransactionObject } from '@switchboard-xyz/solana.js';

    const connection = new Connection("https://api.mainnet-beta.solana.com");
    const program = await SwitchboardProgram.fromConnection(connection);

staticfromProvider

  • fromProvider(provider: AnchorProvider, programId?: any, attestationProgramId?: any): Promise<SwitchboardProgram>

  • Create and initialize a SwitchboardProgram connection object.

    Parameters

    • provider: AnchorProvider

      The anchor provider containing the RPC and wallet connection.

    • optionalprogramId: any
    • optionalattestationProgramId: any

    Returns Promise<SwitchboardProgram>

    The SwitchboardProgram used to create and interact with Switchboard accounts.

    Basic usage example:

    import * as anchor from "@coral-xyz/anchor";
    import { Connection } from "@solana/web3.js";
    import { AnchorWallet, SwitchboardProgram, TransactionObject } from '@switchboard-xyz/solana.js';

    const connection = new Connection("https://api.mainnet-beta.solana.com");
    const provider = new AnchorProvider(
       connection,
       new AnchorWallet(payerKeypair ?? SwitchboardProgram._readOnlyKeypair),
       { commitment: 'confirmed' }
     );
    const program = await SwitchboardProgram.fromProvider(provider);

    const txn = new TransactionObject(program.walletPubkey, [], []);
    const txnSignature = await program.signAndSend(txn);
